## üèÜ AWS Certifications

### 1. AWS Certified Cloud Practitioner (CLF-C02)

**Level**: Foundational  
**Best Time to Take**: After Phase 2  
**Preparation Time**: 1-2 weeks  
**Cost**: $100  
**Valid**: 3 years

**Exam Details**:
- Duration: 90 minutes
- Questions: 65 (multiple choice/multiple response)
- Passing Score: 700/1000
- Format: Online or test center

**Topics Covered**:
1. Cloud Concepts (24%)
   - Cloud computing benefits
   - Economics of cloud
   - Cloud architecture principles

2. Security and Compliance (30%)
   - AWS shared responsibility model
   - Security best practices
   - Compliance programs

3. Cloud Technology and Services (34%)
   - Core AWS services
   - Categories of services
   - Service selection

4. Billing, Pricing, and Support (12%)
   - Pricing models
   - Account structures
   - Support resources

**Study Resources**:
- [AWS Cloud Practitioner Essentials](https://aws.amazon.com/training/digital/aws-cloud-practitioner-essentials/) (Free)
- [AWS Skill Builder](https://explore.skillbuilder.aws/) (Free practice exams)
- [AWS Whitepapers](https://aws.amazon.com/whitepapers/)
- Practice exams on Udemy or Tutorials Dojo

**Study Plan** (1-2 weeks):
```
Week 1: 
- Complete AWS Cloud Practitioner Essentials course
- Read AWS Overview whitepaper
- Review core services

Week 2:
- Take practice exams (aim for 80%+)
- Review weak areas
- Schedule and take exam
```

## üìã Certification Study Strategy

### General Tips

1. **Hands-On Practice** (Most Important!)
   - Complete all labs in learning path
   - Build real projects
   - Use Free Tier/credits

2. **Take Practice Exams**
   - Start 2 weeks before exam
   - Aim for 80%+ consistently
   - Review wrong answers

3. **Read Documentation**
   - AWS FAQs for major services
   - Azure documentation
   - Whitepapers and case studies

4. **Join Study Groups**
   - Reddit: r/AWSCertifications, r/AzureCertification
   - Discord communities
   - LinkedIn groups

5. **Time Management**
   - Create study schedule
   - 1-2 hours daily
   - Consistency is key

### Exam Day Tips

1. **Before the Exam**
   - Good night's sleep
   - Arrive early (or test system for online)
   - Review key concepts
   - Stay calm and confident

2. **During the Exam**
   - Read questions carefully
   - Flag uncertain questions
   - Eliminate wrong answers
   - Watch your time
   - Review flagged questions

3. **Online Exam Setup**
   - Clean workspace
   - Stable internet
   - Valid ID ready
   - Webcam and microphone working
   - No distractions
